# How to Screenshot the Banner Mockup

## Quick Steps

1. **Open the file:**
   - File: `marketing/godgpt-banner-screenshot-ready.html`
   - Already opened in your browser

2. **Take a screenshot:**

   ### On macOS:
   - **Full screen:** Press `Cmd + Shift + 3`
   - **Selected area:** Press `Cmd + Shift + 4`, then drag to select
   - **Window:** Press `Cmd + Shift + 4`, then press `Spacebar`, click the browser window

   ### Recommended Method:
   - Press `Cmd + Shift + 4`
   - Press `Spacebar` to switch to window capture mode
   - Click on the browser window
   - Screenshot saved to Desktop

3. **For best results:**
   - Make sure browser is in full-screen or large window
   - Wait for animations to settle
   - Ensure the banner at the bottom is fully visible

## Alternative: Browser DevTools Screenshot

1. Open DevTools (`Cmd + Option + I`)
2. Press `Cmd + Shift + P` (Command Palette)
3. Type "screenshot" and select:
   - **"Capture full size screenshot"** - for entire page
   - **"Capture node screenshot"** - for specific element

## Image Format

The screenshot will be saved as PNG, which is perfect for proposals.

## Tips

- The mockup is optimized for screenshots (no scrollbars, fixed height)
- All elements are visible in a single viewport
- The banner is prominently displayed at the bottom
- Colors and gradients are optimized for presentation
