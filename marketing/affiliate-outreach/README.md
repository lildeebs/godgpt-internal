## Usage Guidelines

This professional template is designed for all affiliate outreach communications. You can customize the opening line to be more personal based on how closely you've been following the creator's content.

## Customization Checklist

Before sending, make sure to:
- [ ] Replace `[Creator Name]` with actual creator name
- [ ] Replace `[INSERT LINK]` with actual ambassador program link
- [ ] Personalize opening line based on specific content you've seen
- [ ] Review for tone alignment with creator's brand
- [ ] Check that all placeholders are filled

**Note:** All emails are already signed with:
- Dionne
- Creator Partnerships Manager
- GodGPT

## Best Practices

1. **Personalization is Key**: Always reference specific content you've seen
2. **Authenticity Matters**: Match the tone to the creator's brand
3. **Value First**: Lead with what they get, not what you want
4. **Follow Up**: If no response in 1 week, send a gentle follow-up
5. **Track Responses**: Keep notes on which creators respond and what resonates

## Next Steps

After sending initial outreach:
1. Wait 5-7 days for response
2. If no response, send a gentle follow-up
3. Track which creators respond and what works
4. Iterate based on response rates
