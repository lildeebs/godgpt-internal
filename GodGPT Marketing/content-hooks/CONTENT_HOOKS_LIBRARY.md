## Hook Framework Overview

### The 3-Second Rule
**If it doesn't hook in 3 seconds, it's dead.**

Every hook must:
- ✅ Capture attention immediately (0-3 seconds)
- ✅ Create curiosity gap or pattern interrupt
- ✅ Promise value or reveal something interesting
- ✅ Stop the scroll

### Success Metrics
- **Completion Rate Target:** >80%
- **Engagement Rate Target:** >5%
- **Share Rate Target:** >2%
- **Save Rate Target:** >3%

---

## Hook Types (6 Core Categories)

Based on analysis of top-performing videos, these 6 hook types consistently drive high engagement:

1. **Pattern Interrupt** - Unexpected visual or audio
2. **Curiosity Gap** - "The thing nobody tells you about..."
3. **Question Hook** - "POV: You just discovered..."
4. **Emotional Hook** - Immediate emotional reaction
5. **Visual Hook** - Striking first frame
6. **Audio Hook** - Trending sound or unexpected audio

## Hook Testing Framework

### A/B Testing Priority
1. **Test Hook Type:** Same content, different hook type
2. **Test Hook Intensity:** Mild vs Strong emotional trigger
3. **Test Hook Length:** Short (3s) vs Extended (5s)
4. **Test Visual Style:** Minimal vs Bold text overlay
5. **Test Audio:** Trending vs Original

### Success Indicators
- **Hook Works:** >80% completion rate
- **Hook Needs Work:** <60% completion rate
- **Hook is Perfect:** >85% completion + >5% engagement

### Iteration Strategy
- If completion <60%: Change hook type entirely
- If completion 60-80%: Adjust intensity or visual
- If completion >80%: Optimize for engagement (comments/shares)

## Quick Reference: Hook Selection Guide

### Choose Pattern Interrupt When:
- You have unexpected result/revelation
- You want to break viewer expectations
- You have dramatic before/after

### Choose Curiosity Gap When:
- You have hidden information/secret
- You want to create mystery
- You have insider knowledge

### Choose Question Hook When:
- You want direct engagement
- You're using POV format
- You want viewer to self-identify

### Choose Emotional Hook When:
- You have powerful transformation story
- You want deep connection
- You have relatable struggle

### Choose Visual Hook When:
- You have striking visual element
- You want to stop scroll immediately
- You have clear comparison

### Choose Audio Hook When:
- You have trending sound opportunity
- You want pattern interrupt via audio
- You're following current trend

## Production Notes

### Hook Execution Checklist
- [ ] Hook appears in first 3 seconds
- [ ] Visual element is striking/clear
- [ ] Audio supports hook (or creates pattern interrupt)
- [ ] Text overlay is readable (if used)
- [ ] Hook promises value that content delivers
- [ ] Hook creates curiosity that content answers

### Common Mistakes to Avoid
- ❌ Hook doesn't match content (clickbait)
- ❌ Hook is too long (>5 seconds)
- ❌ Visual is unclear or cluttered
- ❌ Audio doesn't support hook
- ❌ Hook doesn't create curiosity or pattern interrupt
- ❌ Text overlay is unreadable
